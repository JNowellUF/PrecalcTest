Notating Sets: how we write domains, codomains, and ranges.

An important piece of the mathematical language is properly writing sets, and there are a number of ways to do this. One such way, which we will outline below, is the so-called ‘set-builder notation’. This is essentially building a set by listing the properties that each element of the set has.
