The most common algebraic description of absolute value that people know, tends to be that it is the thing that “makes the value positive.” Although this is not incorrect, it is a bit imprecise. In this section we discuss how absolute values make something positive, and how one should go about algebraically manipulating absolute values.

Firstly, there is no magic wand in mathematics that simply makes anything positive (or, perhaps we should say, absolute value is the magic wand... so if we want to dive into the mechanics, we need a different tool!)
